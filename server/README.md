## Starting dev. server

1. Install python modules

	`pip install -r requirements.txt`

2. Customize configutaion file `config.py`

3. Change host & port in run.py

	```python
if __name__ == '__main__':
    app.run(host="0.0.0.0", port="3389")
	```

4. Run server

	`python3 run.py`
